# Vaaman's Documentation

Uses sphinx and rst.

## Compile and View

```
make html
$BROWSER build/html/index.html
```

## Edit

`source/index.rst` is the index file, it contains the table of content. Open
index.html and index.rst side-by-side and compare for better understanding.
